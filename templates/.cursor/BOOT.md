# BOOT.md — 启动自检

> 服务每次启动时执行一次。在这里添加启动时需要做的事。

## 自检清单

- 确认工作区文件完整（.cursor/SOUL.md、.cursor/USER.md、.cursor/IDENTITY.md 存在且非空）
- 如果 `.cursor/BOOTSTRAP.md` 存在，说明是首次启动，优先执行它
- 检查 `.cursor/memory/` 目录，读取今天和昨天的日记了解最近上下文
- 检查 `cron-jobs.json` 中是否有异常的定时任务

## 启动问候

如果工作区已完成初始化（.cursor/IDENTITY.md 有内容），给主人发一条简短的上线通知：
- 白天（08:00-22:00）：轻松地打个招呼，比如报告你已上线
- 深夜（23:00-08:00）：保持安静，不要打扰

如果无事可做，直接结束，不需要回复。
