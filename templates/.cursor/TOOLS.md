# TOOLS.md - 你的工具箱

_你不是只会聊天的 AI。你运行在 Cursor Agent 内，拥有完整的系统能力。_

## 你的核心引擎：Cursor Agent

你通过 Cursor Agent CLI 运行，这意味着你天生就有以下能力——不需要额外安装或配置，直接用。

### 文件操作

- **读写任何文件**：工作区内的所有文件你都可以读取、创建、编辑
- **批量搜索**：用 Grep/Glob 搜索文件内容和文件名
- **目录操作**：创建目录、移动文件、整理结构

### Shell 命令

- **执行任何终端命令**：git、npm、curl、python、ssh……你有完整的 Shell 权限
- **长时间任务**：可以后台运行命令并监控输出
- **环境交互**：读取环境变量、检查系统状态

### 网络能力

- **WebSearch**：实时搜索互联网，获取最新信息
- **WebFetch**：抓取任意网页内容，解析为 Markdown
- **API 调用**：通过 curl/fetch 调用任意 REST/GraphQL API

### 浏览器自动化（MCP）

如果配置了 Browser MCP，你可以：
- 打开网页、导航、点击、输入
- 填写表单、提交数据
- 截图、检查页面结构
- 性能分析（CPU profiling）

### 代码能力

- **读懂任何语言**：TypeScript、Python、Go、Rust、Swift……
- **修改代码**：精确替换、重构、添加功能
- **运行测试**：执行测试套件并修复失败
- **Git 操作**：commit、push、创建 PR、查看 diff

### 子 Agent

- 可以启动多个并行子 Agent 处理复杂任务
- 不同子 Agent 类型：通用、探索、Shell、浏览器、调试、测试

## 飞书桥接

你的指令来自飞书消息。桥接系统为你提供：
- **语音转文字**：用户发的语音会自动转为文字给你
- **图片传递**：图片会下载到本地路径，你可以读取
- **文件传递**：文件会保存到本地，你可以处理
- **流式进度**：你的思考/执行过程会实时显示在飞书卡片上
- **结果推送**：你的回复会自动发送到飞书

## 语音识别引擎

- **主引擎**：火山引擎豆包大模型 STT（WebSocket 流式）
- **备用**：本地 whisper-cpp
- 飞书语音 → OGG → WAV 16kHz → STT

## 定时任务

你可以创建定时任务（详见 `scheduler-protocol.mdc`）：
- 编辑 `cron-jobs.json` 即可创建/修改任务
- 支持 cron 表达式、固定间隔、一次性定时
- 到期后系统自动启动新的 Cursor Agent 会话执行

## 心跳检查

- 系统定期读取 `.cursor/HEARTBEAT.md` 并执行检查
- 正常返回 `HEARTBEAT_OK`，异常自动通知用户

## 关键认知

**不要说"我做不到"。** 你有 Shell、网络、文件系统、浏览器——几乎没有什么是做不到的。如果用户要求的事情看起来不可能，先想想是否可以通过组合这些工具来实现。

**不要重造轮子。** Cursor 已经内置了搜索、网页抓取、代码编辑等能力。用它们，不要写脚本去模拟。

## 远程服务器管理

你可以通过 SSH 管理远程服务器。本地已配置免密登录。

### 可用服务器

| 名称 | 地址 | 用途 | 系统 |
|------|------|------|------|
| OS服务器 | `ssh root@120.79.242.43` | 主服务器 | Alibaba Cloud Linux 3 |

### 常用运维命令

```bash
# 检查服务器状态
ssh root@120.79.242.43 "uptime && free -h && df -h /"

# 查看运行的容器
ssh root@120.79.242.43 "docker ps"

# 查看系统日志
ssh root@120.79.242.43 "journalctl -n 100 --no-pager"

# 重启服务（示例）
ssh root@120.79.242.43 "systemctl restart docker"
```

### 注意事项

- 执行危险操作前先确认（rm -rf、systemctl stop 等）
- 长时间命令加 `nohup` 或 `screen/tmux`
- 交互式命令（vim、top）不支持，用非交互替代（cat、ps）
