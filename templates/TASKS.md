# 定时任务说明

> 你可以通过飞书告诉 AI 设定定时任务，AI 会自动创建。
> 也可以直接编辑工作区的 `cron-jobs.json` 文件来管理任务。

## 我的定时任务

暂无。在飞书中发送类似以下消息即可创建：

- "每天早上9点帮我检查邮件"
- "每小时检查一次服务状态"
- "明天下午3点提醒我开会"
- "每个工作日早上汇报昨天的工作"

## 管理指令

| 飞书指令 | 说明 |
|----------|------|
| `/任务` | 查看所有定时任务 |
| `/任务 暂停 ID` | 暂停某个任务 |
| `/任务 恢复 ID` | 恢复某个任务 |
| `/任务 删除 ID` | 删除某个任务 |
| `/任务 执行 ID` | 手动立即执行某个任务 |
| `/心跳` | 查看心跳状态 |
| `/心跳 开启` | 开启心跳检查 |
| `/心跳 关闭` | 关闭心跳检查 |
| `/心跳 执行` | 立即执行一次心跳检查 |
| `/心跳 间隔 分钟数` | 设置心跳间隔（如 `/心跳 间隔 30`） |
